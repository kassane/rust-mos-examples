# rust-mos-examples

Rust examples targeting MOS 6502 platforms via [rust-mos](https://github.com/kassane/rust-mos-examples/releases) and [llvm-mos-sdk](https://github.com/llvm-mos/llvm-mos-sdk).

## Requirements

- [rust-mos-bootstrap](https://github.com/kassane/rust-mos-examples/releases) — Rust toolchain with LLVM-MOS backend.
- [llvm-mos-sdk](https://github.com/llvm-mos/llvm-mos-sdk/releases) v23.x

> ![NOTE]
> Both toolchains are LLVM23 based to LTO.
